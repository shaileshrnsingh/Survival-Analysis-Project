# Survival-Analysis-Project
# Understanding Addiction Relapse Using Survival Analysis

This project focuses on analyzing addiction relapse patterns using Survival Analysis techniques such as Kaplan-Meier Estimator, Log-Rank Test, and Cox Proportional Hazards Regression. The study was conducted using rehabilitation center data to identify significant factors affecting relapse and understand relapse behavior over time.

## Objectives

* Estimate relapse probability using Kaplan-Meier Estimator
* Compare relapse rates between different groups using Log-Rank Test
* Identify significant predictors affecting relapse using Cox Regression
* Suggest possible intervention strategies for rehabilitation centers

## Techniques Used

* Survival Analysis
* Kaplan-Meier Estimation
* Log-Rank Test
* Cox Proportional Hazards Model
* Data Preprocessing
* Statistical Validation

## Tools & Libraries

* R Programming
* survival
* survminer
* ggplot2
* dplyr
* lmtest
* caret

## Dataset

The dataset was collected with support from Muktangan Rehabilitation Center and includes information related to addiction relapse, demographic variables, employment status, qualification, annual income, and relapse duration.

> Note: Original patient-sensitive data has not been uploaded for privacy and ethical reasons.

## Key Findings

* Students showed faster relapse compared to employed individuals
* Isolation significantly increased relapse risk
* Drug addiction cases relapsed faster than alcohol addiction cases
* Critical relapse windows were identified for post-treatment monitoring
